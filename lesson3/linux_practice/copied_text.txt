Git — это система для хранения истории изменений кода и совместной работы.
Linux — ОС, где всё есть файл. Конфиги лежат в /etc, программы — в /bin и /usr/bin.
Команды:
ssh — удаленный доступ к другому компу.
git clone — скачать проект себе.
git status — проверить, что изменилось.
git add — выбрать файлы для сохранения.
git commit — сохранить изменения в истории.
git push — отправить изменения в сеть.
